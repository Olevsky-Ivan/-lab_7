# lab_7
# Приклади форматування та логічних операцій на C

Ця програма демонструє роботу з форматуванням виводу для різних типів даних, використання логічних і побітових операцій, а також введення та порівняння чисел користувачем. Вона показує, як використовувати різні специфікації форматування printf для типів long, int, char і double, включаючи десятковий, шістнадцятковий, вісімковий формати та відображення у науковій нотації.

Програма обчислює індивідуальний вираз a = b + c - '0' та демонструє побітову операцію a ^ ~b. Додатково оголошуються змінні з індивідуального завдання, наприклад u = 0.58797e5 та s = 0xb12, і показуються приклади виводу цих змінних у різних форматах, включаючи %d, %i, %x, %X, %#X, %o, %c для long і %f, %.2f, %e, %E, %g, %G для double.

Програма також демонструє специфікації для інших типів, таких як int і double, на прикладі змінних n = 83 та x = 123.456. Потім виконуються приклади логічних і побітових операцій, включаючи порівняння, кон'юнкцію, диз'юнкцію та заперечення.

На завершення реалізовано введення користувачем двох дійсних чисел та визначення, яке з них менше, із виводом результату у зручному форматі. Програму можна компілювати за допомогою gcc і запускати у консолі. Приклад можливого виводу включає значення обчислених виразів, форматований вивід змінних, результати логічних і побітових операцій, а також порівняння введених чисел.
